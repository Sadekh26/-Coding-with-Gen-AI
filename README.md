# -Coding-with-Gen-AI
## Problem
Design and implement a music playlist manager using Object-Oriented Programming (OOP) and a Linked List data structure. The system should allow users to add, remove, search, and display songs in a playlist, where each song is represented as an object and stored in a custom linked list implementation.
